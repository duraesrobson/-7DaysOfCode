#Day2: Variáveis

Este desafio, consiste em criar um programa que faça três perguntas ao usuário: qual o nome, quantos anos ele tem e qual linguagem de programação está estudando. As respostas serão registradas e, ao final, o programa exibirá uma mensagem personalizada com essas informações.








#Day3: Fluxo de decisão

O desafio é criar um jogo onde o usuário decide seu caminho na programação:

1. Primeiro, ele escolhe se quer seguir para Front-End ou Back-End.
2. No Front-End, pode optar por aprender React ou Vue. No Back-End, as opções são C# ou Java.
3. Depois, ele decide se quer se especializar na área escolhida ou se tornar um desenvolvedor Fullstack, com uma mensagem específica para cada escolha.
4. Por último, ele pode dizer quais tecnologias quer aprender. Ele pode adicionar quantas quiser, uma por vez, enquanto continuar respondendo "ok" para a pergunta "Tem mais alguma tecnologia que você gostaria de aprender?". Cada tecnologia escolhida recebe um comentário especial.

    #Day4: Adivinhe o número

    O desafio é criar um programa que inicia com um número pré-definido entre 0 e 10. O jogador deve adivinhar esse número em até três tentativas. Ao acertar, o usuário é parabenizado, caso contrário, ao final das tentativas, o programa revela o número correto.



    #Day5: Arrays e coleções

    No dia 5 o desafio foi fazer um programa que solicita ao usuário se deseja adicionar uma comida à lista de compras, permitindo respostas "sim" ou "não". Caso a resposta seja positiva, o usuário informa o nome do alimento e seleciona uma categoria predefinida, como frutas, laticínios, congelados ou doces. Quando o usuário decide não adicionar mais itens, o programa exibe a lista organizada por categorias.

    #Day7: Funções em Javascript

    O desafio propõe a criação de uma calculadora simples, mas com um detalhe importante: cada operação deve ser uma função separada dentro do código.

    Primeiro, o programa precisa exibir as opções disponíveis: soma, subtração, multiplicação, divisão e a opção de sair.

    Depois, a pessoa escolhe a operação desejada e informa dois números. O programa faz o cálculo e mostra o resultado na tela.

    Se a opção escolhida for "sair", o programa encerra a execução e exibe a mensagem: "Até a próxima".
