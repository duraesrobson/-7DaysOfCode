#Day2: Variáveis

Este desafio, consiste em criar um programa que faça três perguntas ao usuário: qual o nome, quantos anos ele tem e qual linguagem de programação está estudando. As respostas serão registradas e, ao final, o programa exibirá uma mensagem personalizada com essas informações.








#Day3: Fluxo de decisão

O desafio é criar um jogo onde o usuário decide seu caminho na programação:

1. Primeiro, ele escolhe se quer seguir para Front-End ou Back-End.
2. No Front-End, pode optar por aprender React ou Vue. No Back-End, as opções são C# ou Java.
3. Depois, ele decide se quer se especializar na área escolhida ou se tornar um desenvolvedor Fullstack, com uma mensagem específica para cada escolha.
4. Por último, ele pode dizer quais tecnologias quer aprender. Ele pode adicionar quantas quiser, uma por vez, enquanto continuar respondendo "ok" para a pergunta "Tem mais alguma tecnologia que você gostaria de aprender?". Cada tecnologia escolhida recebe um comentário especial.
